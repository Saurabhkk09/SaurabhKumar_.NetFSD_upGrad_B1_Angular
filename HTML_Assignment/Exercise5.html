Technical Terms (Definition List)

HTML
HyperText Markup Language, the standard language for creating web pages.
Element
A component of an HTML document, consisting of a start tag, content, and an end tag.
Attribute
Provides additional information about an HTML element, usually appearing in name/value pairs.
